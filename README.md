# 🩺 Breast Cancer Classification using Artificial Neural Network (ANN)

A Deep Learning project that classifies breast tumors as **Benign** or **Malignant** using an Artificial Neural Network (ANN) built with TensorFlow/Keras.

---

## 📌 Project Overview

Breast cancer is one of the most common cancers worldwide, and early diagnosis plays a vital role in improving treatment outcomes.

This project uses an **Artificial Neural Network (ANN)** to classify breast tumors based on 30 diagnostic features from the Breast Cancer Wisconsin Diagnostic Dataset.

---

## 🎯 Objective

The objective of this project is to develop a Deep Learning model capable of accurately classifying breast tumors into:

- ✅ Benign (Non-Cancerous)
- ❌ Malignant (Cancerous)

using the Breast Cancer Wisconsin Diagnostic Dataset.

---

## 📂 Dataset Information

- **Dataset:** Breast Cancer Wisconsin Diagnostic Dataset
- **Source:** Scikit-learn
- **Total Samples:** 569
- **Number of Features:** 30
- **Classes:** 2

| Label | Class |
|-------|--------|
| 0 | Malignant |
| 1 | Benign |

## 🧠 Deep Learning Model Architecture

The Artificial Neural Network consists of:

- Flatten Layer
- Dense Hidden Layer (20 Neurons, ReLU)
- Output Layer (2 Neurons, Softmax)
